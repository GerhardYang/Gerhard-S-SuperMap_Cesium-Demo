<h3 id="example-1">目录结构</h3>

范例目录结构：
``` 
            ./
              └─3dwebgl         3dwebgl范例目录
                  ├─img             缩略图目录
                  └─config.js       配置文件  
              ├─Cesium       SuperMap Cesium库
              ├─css         公共css目录
              ├─fonts       公共字体目录
              ├─img         公共图片目录
              ├─js          公共js目录
              ├─libs        公共依赖
              ├─locales     语言包
              │─template    界面模板
              
  
  
```
